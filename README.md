# DiscordStatusChanger

> [!CAUTION]
> This is technically a selfbot which is against Discord's TOS, so use with caution.  
> However, there are usually no issues with selfbotting for non-malicious purposes such as changing your status.

## Commands
- `change-status`: The main command for changing your Discord status
  - `--token` (`string`): Your Discord token (retrieve this using this method: [How To Get Your Discord Token From the Browser Console](https://gist.github.com/MarvNC/e601f3603df22f36ebd3102c501116c6/2831d42c40602157d1f313c2dc880fc5fc50a460))
  - `--status` (`string`): The status you would like to set
